# EnglishLessonBot
Бот-репетитор з англійської з флеш-картками
Telegram-бот, який зберігає у БД словниковий запас користувача, надсилає за командою encard йому 5 нових слів з перекладом і через годину перевіряє засвоєння у форматі вікторини. Веде статистику правильних і неправильних відповідей. Демонструє роботу з розкладом задач і збереженням стану діалогу.

Стек: PostgreSQL на EC2 + aiogram
